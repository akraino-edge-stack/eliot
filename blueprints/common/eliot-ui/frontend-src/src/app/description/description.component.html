ELIOT Portal Description
What is main purpose of ELIOT Portal ?

ELIOT Portal helps users deployment easy by providing ELIOT cluster which includes unified development environment which internally uses Kubernetes cluster (k8s master + worker nodes), docker, EdgeX, OPC-UA, Cadvisor, etc

ELIOT Portal mainly provides an efficient way to individuals in Edge computing domain making the development environment as developer friendly.
